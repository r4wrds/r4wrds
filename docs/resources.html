Lesson Datasets

  • Full Dataset as a zipped folder. Inside you’ll find these files:
    • Longitude-Latitude Data: latlon.csv
    • CSCI Data: cscidat.csv
    • ASCI Data: ascidat.csv
  • Spatial Data (generated using this lesson ) is available as a zipped folder here. This folder contains:
    • states_boundaries.shp (along with associated .prj, .dbf, .shx)
    • states_boundaries.kml A google earth kml file
    • ca_counties_boundaries.geojson A geojson of counties in CA
